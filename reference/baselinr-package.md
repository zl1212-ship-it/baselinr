# baselinr: WWC-aligned baseline equivalence for education impact evaluations

baselinr produces report-ready baseline equivalence tables for impact
evaluations in education research, following the conventions of the What
Works Clearinghouse (WWC). It takes a data frame, a treatment indicator,
and a set of covariates, and reports — for each covariate — the
appropriate standardized effect size (Hedges' g for continuous
covariates, the Cox index for binary covariates) together with the WWC
baseline-equivalence category.
